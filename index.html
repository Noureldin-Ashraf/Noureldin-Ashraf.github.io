Here are some of the projects i have created.

Projects which aimed to increase productivity and enhance processes.

Content Managment System

A two part System. The first part is integrated with Microsot rms giving the ability to:
- Create new product on both systems,
- Manipulate existing data like (price, cost, supplier, components....etc),
- Create and cancel promotions on products
- Enable/ Disable products.
The Second part is a full integration portal to send all the actions mentioned above to a Magento based website using set of APIs. Achieving a complete seemless integration among the two systems

HandHeld Collating App

Mobile APP on a Handheld device integrated with microsoft RMS to retrieve order data and organize it based on a set of preset criteria. Orders then are either auto assigned to Collator(s) based on order capacity and products count in each order or overridden and assigned manually by the supervisors. The collator can then start preparing the order by scanning the products on the APP to mark them as collated. The collator can also do a number of other operations like (Removing product from order - Replace product with another - Divide item on multiple lines ...etc) all of this actions are authorized per user group. lastly, when the order is done it can be cashed and a reciept is printed and this is reflected directly on RMS. Attached to it a Web backend module to manage: User accounts, Preparation lead time and API logs

Order Monitoring Dashboard

Dashboard for the top managment to monitor the order volume accross each store/location realtime showing: pending orders, completed orders and late orders. Giving the ability to take preemptive actions.
The Dashboard also shows historical order data for previous days, weeks, months and years for comparison. All the mentioned reports are created using Python Matplotlib and Seaborn for data visualization.

Order Preparation Screen

System to manage incoming orders and distribute each product to its designated department/ station. It also monitor and broadcast each department status to the main consolidated screen ex:(pending preparation, finished preparing, have an out of stock item ....etc.). This system also has potentiol application in resturant buisness. Attached to it a backend module to manage: User accounts, Preparation lead time and Product Category assigned per each department/station.

Customer Relationship Management

CRM System that gives the ability manage and manipulate customer data and also the ability to create and monitor orders for the customers.
it also gives the ability to monitor order preparation process at stores. additional layer of integration is made between the CRM system and a Magento based website to integrate and link web orders to customers.
